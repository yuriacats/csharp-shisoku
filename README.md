# csharp-shisoku
このリポジトリーは、オリジナルプログラミング言語を作っているリポジトリーです（？）

## How To Use

```terminal
$ dotnet run --project shisoku
```
以下のコマンドで対話形式で計算することができます。
使うことが可能な演算子は `+ - * / ()` です。

上記の演算子、数字以外を入力した場合はErrorを返します。
